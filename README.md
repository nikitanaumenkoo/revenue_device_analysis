# Revenue & Device Analysis (SQL)
## Опис проєкту
SQL-скрипт формує аналітичний датасет для оцінки виручки та активності користувачів по континентах та типах пристроїв.
Проект дозволяє аналізувати загальну виручку, виручку з мобільних та десктопних пристроїв, кількість акаунтів, підтверджених акаунтів та сесій, що допомагає приймати бізнес-рішення по регіонах та оптимізації каналів продажу.
## Джерела даних
- `order` — інформація про замовлення  
- `product` — товари та їх ціни  
- `session_params` — параметри сесій (континент, пристрій)  
- `account` — інформація про акаунти  
- `account_session` — зв’язок акаунтів із сесіями
## Реалізована логіка
- Розрахунок загальної виручки по континентах  
- Розподіл виручки за типами пристроїв (mobile / desktop)  
- Підрахунок кількості акаунтів та підтверджених акаунтів  
- Підрахунок кількості сесій по континентах  
- Об’єднання метрик у фінальний аналітичний датасет  
- Розрахунок частки виручки від загальної (`percent_Revenue_from_Total`)
## Ключові метрики
- Загальна виручка  
- Виручка з мобільних пристроїв  
- Виручка з десктопних пристроїв  
- Частка виручки від total  
- Кількість акаунтів  
- Кількість підтверджених акаунтів  
- Кількість сесій  
## SQL-інструменти
- CTE (WITH)  
- JOIN / LEFT JOIN  
- CASE WHEN (умовна агрегація)  
- GROUP BY  
- Агрегаційні функції (SUM, COUNT)
